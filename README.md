# Projekt-Inzynieria-Oprogramowania

## Opis projektu

Zawarte w tym repozytorium komponenty i częsci projektu są z zakresu Inżynierii Oprogramowania. Celem tego projektu jest gromadzenie kodu źródłowego, dokumentacji, diagramów oraz innych materiałów ważnych podczas procesu tworzenia oprogramowania.

Tematem tego projektu jest aplikacja klubu sportowego wschodniej sztuki walki. Aplikacja ta ma ułatwić użytkownikom załatwiania klubowych spraw takich jak płatności, czy zapisy na zawody oraz ułatwi informowanie zawodników i ich rodziców o nowościach dziejących się w klubie.
